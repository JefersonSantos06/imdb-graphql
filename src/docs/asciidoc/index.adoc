API de wrapper/crawler do IMDB. Dado um código, retorna os dados de um filme, de
uma pessoa ou o calendário de estreias. As respostas são cacheadas por (tipo,
código, locale) com TTL configurável.

== Autenticação

Todos os endpoints (exceto `/actuator/health` e esta documentação) exigem
autenticação *HTTP Basic*. Envie o cabeçalho `Authorization: Basic <credenciais>`
em cada requisição. Sem credenciais válidas a API responde `401 Unauthorized`.

== Convenções

* `locale` (query, opcional): idioma/região dos campos localizados. Default: `pt-BR`.
* `ttl` (query, opcional): validade do cache em segundos; `0` força um novo scrape.
* Códigos: filme = `ttNNNNNNN`, pessoa = `nmNNNNNNN`. Formato inválido retorna `400`;
  código inexistente retorna `404`.
